# 🧠 Agentic AI Customer Review Analyzer

An AI-powered NLP application that automatically analyzes customer reviews using Sentiment Analysis, Emotion Detection, Complaint Detection, Topic Modeling, Semantic Search, and AI-based Business Insights.

Built with **Python, Streamlit, Transformers, Sentence Transformers, FAISS, Scikit-learn, and LangChain**, this application helps businesses understand customer feedback and make data-driven decisions.

## 📌 Features

- ✅ Customer Review Upload (CSV)
- ✅ Text Preprocessing
- ✅ Sentiment Analysis (Positive / Negative / Neutral)
- ✅ Emotion Detection
- ✅ Complaint Detection
- ✅ Recommendation Generation
- ✅ Topic Modeling (LDA)
- ✅ Semantic Search using FAISS
- ✅ AI-powered Customer Reply Generation
- ✅ Business Summary using LangChain
- ✅ Interactive Charts & Dashboards

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ankitha1830-star/Agentic-AI-Customer-Review-Analyzer.git
```

Move into the project folder

```bash
cd Agentic-AI-Customer-Review-Analyzer
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
